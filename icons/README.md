# Icons for SendToERP Add-in

Иконки для аддина Outlook Web SendToERP в различных размерах.

## Размеры

- **icon-16.svg** - 16x16 пиксели (для панели инструментов)
- **icon-32.svg** - 32x32 пиксели (для дополнительного отображения)
- **icon-80.svg** - 80x80 пиксели (для магазина Office)

## Использование

Эти иконки используются в файле `manifest.xml`:

```xml
<bt:Image id="icon16" DefaultValue="https://ваш-домен.com/icons/icon-16.svg"/>
<bt:Image id="icon32" DefaultValue="https://ваш-домен.com/icons/icon-32.svg"/>
<bt:Image id="icon80" DefaultValue="https://ваш-домен.com/icons/icon-80.svg"/>
```

## Замена иконок

Вы можете заменить эти иконки на свои:

1. Подготовьте три PNG или SVG иконки нужных размеров
2. Заместите файлы в этой папке
3. Обновите URLs в manifest.xml (если используете другой домен)

## Требования к иконкам

- **Формат:** PNG или SVG
- **Цвета:** рекомендуется использовать официальные цвета Microsoft (синий #0078d4)
- **Прозрачность:** поддерживается
- **Размеры:** точно указанные (16x16, 32x32, 80x80)
