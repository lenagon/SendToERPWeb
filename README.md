# SendToERP Web Add-in

Аддин для Outlook Web, который позволяет быстро отправлять данные письма в ERP систему с помощью трёх кнопок на панели инструментов.

## Функциональность

### Три основные кнопки:

1. **Кнопка 1** - Отправить заказ
   - Отправляет содержимое письма как заказ в ERP систему

2. **Кнопка 2** - Отправить счет
   - Отправляет содержимое письма как счет в ERP систему

3. **Кнопка 3** - Отправить отчет
   - Отправляет содержимое письма как отчет в ERP систему

При нажатии на каждую кнопку пользователю показывается уведомление об успешной отправке.

## Установка и развертывание

### Требования

- Microsoft Office 365 подписка
- Outlook Web App (веб-версия Outlook)
- HTTPS сервер для размещения файлов аддина

### Шаги для развертывания

1. **Разместите файлы на HTTPS сервере:**
   - Загрузите все файлы (`taskpane.html`, `taskpane.js`, `styles.css`) на ваш сервер
   - Убедитесь, что сервер работает по HTTPS

2. **Обновите URLs в manifest.xml:**
   ```xml
   <bt:Url id="functionFile" DefaultValue="https://ваш-домен.com/taskpane.html"/>
   ```

3. **В Outlook Web добавьте аддин:**
   - Откройте Outlook Web App
   - Перейдите в параметры (⚙️)
   - Выберите "Get Add-ins" → "My Add-ins" → "Add a Custom Add-in"
   - Выберите "Add from URL..."
   - Вставьте URL вашего `manifest.xml`:
     ```
     https://ваш-домен.com/manifest.xml
     ```

4. **Протестируйте аддин:**
   - Откройте любое письмо в Outlook Web
   - На панели инструментов должны появиться три кнопки
   - Нажмите на любую кнопку, чтобы увидеть уведомление

## Использование

После установки аддина:

1. Откройте письмо в Outlook Web
2. На панели инструментов вы увидите группу "Отправить в ERP" с тремя кнопками
3. Нажмите на нужную кнопку для отправки данных в ERP систему
4. Вы получите подтверждение в виде зелёного уведомления

## Модификация

### Изменение названий кнопок

Отредактируйте `manifest.xml`:

```xml
<bt:String id="button1Label" DefaultValue="Ваше название"/>
```

### Изменение сообщений уведомления

Отредактируйте `taskpane.js`:

```javascript
function onButton1Click() {
    const message = "✓ Ваше сообщение";
    showNotification(message);
}
```

### Добавление иконок для кнопок

1. Подготовьте три PNG иконки:
   - 16x16 пиксели
   - 32x32 пиксели  
   - 80x80 пиксели

2. Разместите их на сервере

3. Обновите URLs в `manifest.xml`:

```xml
<bt:Image id="icon16" DefaultValue="https://ваш-домен.com/icon-16.png"/>
<bt:Image id="icon32" DefaultValue="https://ваш-домен.com/icon-32.png"/>
<bt:Image id="icon80" DefaultValue="https://ваш-домен.com/icon-80.png"/>
```

## Безопасность

- Все коммуникации осуществляются по HTTPS
- HTML код экранируется для предотвращения XSS атак
- Используется официальный Office JavaScript API
- Аддин не требует специальных разрешений
